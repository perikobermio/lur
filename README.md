# LUR

Escaparate web corporativo para Lur Uribarren, guia de montana.

## Desarrollo

```bash
docker compose up lur-web
```

Disponible en `http://localhost:5173`.

## Produccion

```bash
docker compose up lur-web-prod --build
```

Disponible en `http://localhost:8080`.

## Stack

- Vite
- React
- Docker Compose
